Detailed Description

Ball structure for screen display in client resolution.

Structure representing a ball for screen display.

Stores the result of converting a BallObject received from the server into pixel units.

This structure represents a ball object that has been converted from a logical coordinate system to pixel coordinates for display on the client's screen.
